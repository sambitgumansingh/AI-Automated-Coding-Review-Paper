# AI in Automated Coding – Review Paper

## Author
Sambit Gumansingh  
B.Tech CSE, ASBM University  

## Abstract
This paper discusses the advantages and disadvantages of using Artificial Intelligence in automated coding. It analyzes productivity improvements, code quality impact, ethical concerns, and future implications for software developers.

Presented at International Management Conference (IMCON), ASBM University, 2026.
